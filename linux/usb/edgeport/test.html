Linux Inside Out Networks (Digi) Edgeport Drivers

Drivers
The io_edgeport and io_ti drivers in the Linux kernel will work for all Inside Out Networks Edgeport devices.

There are a number of problems in the Linux USB Edgeport driver that was included in older kernel versions. Please upgrade to the latest 2.4.x kernel version to fix all known problems.

If you have multiple Edgeport devices in the system, you must upgrade to the 2.4.19 kernel or later.

The io_edgeportdriver currently supports the following devices from Inside Out Networks:
  • Edgeport/4
  • Edgeport/4t
  • Edgeport/2
  • Edgeport/4i
  • Edgeport/2i
  • Edgeport/421
  • Edgeport/21
  • Rapidport/4
  • Edgeport/8
  • Edgeport/2D8
  • Edgeport/4D8
  • Edgeport/8i
The io_ti driver currently supports the TI based devices from Inside Out Networks which include:
  • Edgeport/1
